# 📊 Vrinda Store Sales Dashboard (Microsoft Excel)

An interactive **Sales Dashboard** created in **Microsoft Excel** to analyze and visualize sales performance using Pivot Tables, Pivot Charts, and Slicers. This dashboard transforms raw sales data into meaningful business insights, enabling users to monitor sales trends, customer behavior, and product performance.

---

## 📌 Project Overview

The dashboard provides a comprehensive analysis of Vrinda Store's sales data by presenting key performance metrics and interactive visualizations. Users can filter data dynamically to gain insights into sales across different months, channels, customer demographics, and product categories.

---

## 🚀 Features

- 📈 Monthly Sales & Order Analysis
- 👥 Customer Analysis by Age and Gender
- 🛍️ Sales by Product Category
- 🌐 Sales Channel Performance
- 🗺️ State-wise Sales Distribution
- 🎛️ Interactive Slicers for Dynamic Filtering
- 📊 Professional Dashboard Layout
